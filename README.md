# 🚀 Mac 自动化工具合集：告别重复劳动！

> 专为Mac用户打造的实用自动化脚本，一键搞定Excel批量处理、文件整理等高频操作，解放双手！

## ✨ 功能介绍
### 1. Excel批量合并工具
- 自动创建「待合并文件」文件夹，无需手动配置
- 支持 `.xlsx` 和 `.xls` 格式文件批量合并
- 自动标记「来源文件」，方便追溯数据
- 合并结果一键导出为 `合并结果.xlsx`

## 🛠️ 使用教程
1.  下载 `excel_batch_merge.py` 脚本
2.  双击运行脚本，会自动创建「待合并文件」文件夹
3.  把要合并的Excel文件放进文件夹
4.  再次运行脚本，等待合并完成即可

## 💻 运行环境
- Mac OS（全版本兼容）
- Python 3.x
- 依赖库：pandas（运行前执行 `pip install pandas` 安装）
